### MCP Servers (Model Context Protocol)
MCP Servers provide standard programmatic capabilities (like searching a local database or a custom tool) to models.
- When scaffolding an MCP server for NodeJS, create a `package.json` to handle `@modelcontextprotocol/sdk`.
- Map the execution command securely to the `gemini-extension.json` mapping parameters like `${extensionPath}`.
- Fetch `docs/tools/mcp-server.md` or `docs/extensions/reference.md` to guarantee accurate syntax.
